== Description ==

PuzzleMe™ by Amuse Labs is the best platform to create and publish smart games like crossword, sudoku, word search, codeword, kriss kross, quiz and many other game types. Use our WordPress plugin to publish your puzzles with just a shortcode. 

Start creating and publishing your puzzles in 3 simple steps :
1. Login to a <a target="_blank" href="https://amuselabs.com/pmm/login">PuzzleMe account</a>.
2. Create a puzzle and copy the WordPress shortcode from the "Preview & Publish" page.
3. Paste the shortcode in your page or post and hit Publish.

== Frequently Asked Questions ==

= Which puzzles do you support? =

We support creating and publishing smart games - crossword, word search, sudoku, codeword, WordroW, jigsaw, kriss kross and quiz, among others. See our <a target="_blank" href="https://amuselabs.com/games/">game pages</a> for a full list of puzzles. 

= Do I need to create an account? =

While you can initially create a puzzle without logging in, you’ll need to log in to edit or manage your puzzle, or save it beyond 30 days. We recommend using your Google or Facebook account to <a target="_blank" href="https://amuselabs.com/pmm/login">login</a>. 

= Do puzzle solvers need to create an account? =

Your puzzle solvers do not need to create an account.
